## 指标分类概述

### 📋 2. 威廉指标 (Williams %R) - **规划中**
- **WR14**: 14日威廉指标
- **WR21**: 21日威廉指标  
- **计算**: (最高价-收盘价)/(最高价-最低价) × 100
- **功能**: 超买超卖信号，与RSI互补验证

## RSI系统实战应用

### 🎯 1. 基础超买超卖策略
- **RSI6 > 80**: 短期强超买，警惕回调
- **RSI6 < 20**: 短期强超卖，关注反弹
- **RSI12 > 70**: 中期超买区域，考虑减仓
- **RSI12 < 30**: 中期超卖区域，考虑加仓
- **RSI24**: 长期趋势判断，过滤短期噪音

### 📊 2. 多周期RSI组合策略
- **RSI6 > RSI12 > RSI24**: 强势上涨趋势
- **RSI6 < RSI12 < RSI24**: 弱势下跌趋势
- **RSI差值 > 15**: 短期与长期背离严重
- **RSI变化率 > 5%**: 动量变化显著

### 📈 3. RSI高级策略
- **顶背离**: 价格新高但RSI不创新高，看跌
- **底背离**: 价格新低但RSI不创新低，看涨
- **RSI突破50**: 多空分界线，趋势确认
- **连续3天RSI12变化率同向**: 动量加强信号

## 📊 数据输出格式

### CSV文件字段

### 字段说明

- `rsi_6`: 6日RSI (范围: 0-100)
- `rsi_12`: 12日RSI (范围: 0-100)  
- `rsi_24`: 24日RSI (范围: 0-100)
- `rsi_diff_6_24`: RSI6与RSI24差值
- `rsi_change_rate`: RSI12日变化率 (%)
- `calc_time`: 计算时间戳
